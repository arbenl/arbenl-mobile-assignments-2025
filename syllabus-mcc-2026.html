01 Hyrje në Cloud & MCC Studenti shpjegon cloud computing, MCC dhe shembuj të përdorimit në aplikacione mobile. AWS SimuLearn: Cloud First Steps Free Ligjëratë, diskutim, reflektim i shkurtër individual.
02 Arkitektura e MCC Studenti lidh shërbimet cloud me arkitekturën e një aplikacioni mobile-cloud. AWS SimuLearn: Cloud Computing Essentials Free Analizë arkitekture, ushtrim në grup, quiz i shkurtër.
03 Cloud Economics Studenti vlerëson koston, zgjedh kompromiset teknike dhe propozon optimizime bazë. AWS SimuLearn: Cloud Economics Free Punë praktike me skenar kostoje dhe diskutim.
04 Networking Fundamentals Studenti përshkruan VPC, subnet, lidhje dhe rrjedhën bazë të trafikut në cloud. AWS SimuLearn: Networking Concepts Free Skicim topologjie, diskutim teknik dhe pyetje kontrolluese.
05 Të dhënat në cloud Studenti dallon kur përdoret NoSQL dhe modelon një skemë të thjeshtë për aplikacion cloud. AWS SimuLearn: First NoSQL Database Free Ushtrim me access patterns dhe modelim të të dhënave.
06 Compute, VM dhe kontenierë Studenti shpjegon compute choices dhe ndërton një propozim të thjeshtë për hostim cloud. AWS SimuLearn: Computing Solutions Free Demo teknike, punë në çift dhe kontroll i rezultateve.
07 Security & Privacy në MCC Studenti aplikon koncepte bazë të IAM, qasjes dhe sigurisë për aplikacione cloud. AWS SimuLearn: Core Security Concepts Free Analizë risku, diskutim mbi shared responsibility dhe quiz.
08 Resilience, auto-healing & scaling Studenti shpjegon scaling, auto-healing dhe disponueshmëri të lartë për workload cloud. AWS SimuLearn: Auto-Healing and Scaling Applications Free Skenar troubleshooting, analizë konfigurimi dhe diskutim i zgjidhjes.
09 High availability për aplikacione web Studenti projekton një aplikacion web më rezistent me load balancing dhe auto scaling. AWS SimuLearn: Highly Available Web Applications Free Analizë arkitekture, diagram dhe pyetje kontrolluese.
10 Hyrje në Generative AI në cloud Studenti shpjegon konceptet kryesore të GenAI dhe përdorimet e tyre në aplikacione cloud/MCC. AWS SimuLearn: Get Started with Generative AI Free Diskutim, demonstrim dhe reflektim i shkurtër mbi rast përdorimi.
11 Amazon Bedrock playgrounds Studenti krahason foundation models dhe vlerëson rezultatet për një skenar biznesi. AWS SimuLearn: Explore the Amazon Bedrock Playgrounds Free Eksperiment me modele, krahasim i rezultateve dhe diskutim.
12 AI-assisted web development Studenti përdor GenAI për të përmirësuar kod dhe e lidh me hostim në cloud. AWS SimuLearn: Generate Code for a Webpage Free Ushtrim praktik, review kodi dhe kontroll funksional.
13 AI services me SageMaker Studenti integron shërbime AI për përpunim dokumentesh, teksti, zëri dhe përkthimi. AWS SimuLearn: Use AI Services with Amazon SageMaker Free Workflow teknik, analizë rezultatesh dhe shënime laboratorike.
14 AI smart assistants Studenti shpjegon si ndërtohet një asistent AI me backend cloud dhe data services. AWS SimuLearn: Create an AI Smart Assistant Free Projektim zgjidhjeje, diagram shërbimesh dhe diskutim sigurie.
15 Enterprise knowledge assistants Studenti shpjegon RAG, knowledge bases dhe përgatit prezantimin final të projektit. AWS SimuLearn: Create an Enterprise Knowledge Assistant Free Prezantim final, demo ose walkthrough teknik dhe reflektim mbi zgjidhjen.
